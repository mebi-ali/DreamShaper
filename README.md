# DreamShaper

**DreamShaper** is an AI program that can modify your body shape based on a given body fat percentage. This tool is designed to provide users with a visual representation of their potential body transformation by simulating changes in body fat.

## Features
- Simulate body shape changes based on body fat percentage.
- Visualize potential body transformations.

## Installation

### Step 1: Create a Virtual Environment
Before running the application, it's recommended to create a virtual environment to manage dependencies.

```bash
python3 -m venv venv
```
Activate the virtual environment 
```bash
source venv/bin/activate
```
### Step 2: Install Dependencies
Install the required packages by running:

```bash
pip install -r requirements.txt
```
### Step 3: Run the Application
```bash
python3 app.py
```
